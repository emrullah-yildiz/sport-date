## Sourcing rules (for future additions)
- **Free commercial-use only** (Pexels / Unsplash license). No "editorial only" images.
- **Adults only.** No minors — this is an adult product.
- Prefer real, candid, active, *diverse* people over polished studio stock.
- Record every new file in this table with its source + photo id.
- Downloaded (not hotlinked) so posts don't break if the CDN URL changes.
