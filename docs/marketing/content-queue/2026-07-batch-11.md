# Content batch 11 — what you can actually DO with KeepItUp

**Author:** social-media-growth strategist · **Created:** 2026-07-06 · **Status:** all `draft`

**Owner directive (2026-07-06):** stop letting people assume "dating app" is the whole story.
Show what KeepItUp actually lets you do — dating, finding a regular crew, organizing any small
group activity (sport is the heart of it, but the activity is free text), keeping every game you
host or join in one list, and the real superpower: post a plan you already have, let strangers
nearby ask to join, and decide who comes. Flagship 8-slide carousel + one supporting checklist
post. Every feature named below was checked against the codebase before writing:

- **Free-text activity, not a fixed sport list** — `CreateEventForm.tsx`'s sport field is a plain
  text input (placeholder "Tennis", 60-character limit, no dropdown); the signup sport picker also
  has "Don't see your sport? Add your own" (`StepSports.tsx`). Chess night, a hike, a dance class —
  all valid.
- **Group size 2–20** — capacity field on event creation, same audited range used in batch 07 P5.
- **Dating / friendship / group intent** — set at signup, editable any time, audited in batch 07 P3.
- **Hosted + joined games in one list** — `/hosting` ("Your events") shows every event a member is
  hosting with join-request and capacity status; framed here honestly as "your games," not a
  general calendar-sync claim.
- **Request-to-join core loop** — a host publishes an event; anyone nearby can request to join;
  the host accepts or declines. This is the same mechanic audited in batch 07 P1/P2/P5, just named
  here as the "outside the box" feature it actually is.
- **No payment/ticketing claim.** Event payment links are a real, tracked feature request
  (`CX-20260706-group-event-payment-links`) waiting on Stripe — **not implemented**. The
  concert/gig slide (7) is written to organize the *meetup* only (a walk, a pre-show run) — it
  never implies buying or checking tickets through KeepItUp.

**Photos used** (all already downloaded, real, Pexels-licensed, adults only — see
`apps/web/public/brand/social/photos/LICENSES.md`). Eight distinct photos across the carousel, a
ninth for the supporting post — none repeated within the batch, and four of the nine
(cyclists-chat, basketball-friends, park-fitness-group, badminton-sunset) haven't appeared as a
composite in any prior batch, which keeps the rotation fresh per the standing "don't reuse the
same photo across many posts" rule.
