Ready-to-send, individually personalized DMs to real, verified sports-community

**Status:** ~zero users today. KeepItUp opens **Friday 2026-07-10**. No DM
below claims traction we don't have — every one says "brand new" / "just

**Tense note:** every DM says "we're opening KeepItUp on July 10." If you
send any of these on or after July 10, change that phrase to "we just
opened KeepItUp" (search-and-replace, 2 seconds) before pasting.

## How to send (owner)

1. Send from **Instagram @keepitup.social**, one organizer at a time, as a
   normal DM (paste the draft, tweak nothing unless it reads unnatural to you).
2. Pace it — **about 5 per day**, spread across the week. Don't send 20 in
   one sitting; it should never look automated or bulk.
3. Log every send in the tracking table at the bottom (date sent, reply
   yes/no, outcome) so we know what to follow up and what's working.

## Follow-up nudge (use once, only if no reply after 5–7 days)

> Hi again! No pressure at all — just wanted to check if you got a chance
> to look at KeepItUp. If it's not useful for [club name], no worries, and
> if you have five minutes for feedback on what would make it useful, I'd
> genuinely take it. Either way, thanks for the great work you do for
> [city]'s [sport] community.

Send this **once only**. If still no reply, stop — don't send a third message.

## Checklist for finding the next batch (so any agent can repeat this)
1. Search `"<city>" "<sport>" community Instagram beginners welcome 2026`.
2. For every candidate, get an independent second source beyond the
   Instagram page itself — a Meetup group page (member count + real
   upcoming 2026 dates), an official club website (footer copyright year,
   named schedule), or a Linktree (bio text + links).
3. Only count it verified if that second source is live and current. A
   dead link, a 404, or a domain that won't resolve = drop it, don't guess.
4. Never invent a handle, follower count, or schedule — write "could not
   verify" instead and move to the next candidate.
